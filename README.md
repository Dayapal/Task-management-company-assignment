# 🚀 Task Management Company Assignment

## 📖 Overview

A full-stack **Task Management System** built using the **MERN Stack** with **Role-Based Access Control (RBAC)**, **JWT Authentication**, **Admin Dashboard**, **Activity Tracking**, and **Analytics Dashboard**.

This application allows users to efficiently manage tasks while providing administrators with complete control over users, tasks, and system activities.

---

## ✨ Key Features

### 🔐 Authentication & Authorization

- User Registration
- User Login
- JWT-Based Authentication
- Protected Routes
- Role-Based Access Control (Admin/User)
- Admin-Only Access

### 👤 User Features

- Create Tasks
- View Own Tasks
- Update Own Tasks
- Delete Own Tasks
- Track Task Status

### 🛡️ Admin Features

- View All Users
- Activate/Deactivate Users
- Delete Users
- View All Tasks
- Delete Any Task
- View Activity Logs
- Access Analytics Dashboard

### 📊 Analytics Dashboard

- Total Users
- Total Tasks
- Completed Tasks
- Pending Tasks
- Task Distribution Visualization

### 📝 Activity Logging

- User Login Activity
- Task Creation Activity
- Task Update Activity
- Task Deletion Activity
